Qt
how to install

Qt Creator - найвідоміша середовище розробки для створення графічних додатків. Якщо ви збираєтеся створити хороший додаток з графічним інтерфейсом, Qt Creator стане для вас ідеальним вибором.

Деякі з функцій Qt Creator:


  • простий і зручний конструктор GUI-форм;
  • кроссплатформенность;
  • підтримка налагодження, компіляції, профілювання, автозаповнення коду і рефакторінга;
  • підтримка аналізу статичного коду;
  • швидкий компілятор Qt;
  • візуалізація даних Qt;
  • Qt Quick 2D Renderer;
  • Qt WebView;
  • Qt Virtual Keyboard.




CLion
how to install

Це відмінне середовище розробки на C ++, створена відомою компанією JetBrains. Вона поставляється з деякими чудовими функціями, такими як «Smart Editor», «Code Analysis», «Embedded Terminal».

CLion - платна програма, яка не має безкоштовної версії, однак ви можете отримати пробну версію протягом 30 днів, щоб перевірити її.

Деякі з можливостей Dev-C ++:

  • мала вага;
  • проста у використанні панель інструментів;
  • автозавершення коду;
  • гарячі клавіші для компіляції і запуску, наприклад F9 і F10 ;
  • проста установка.

Visual Studio

How to install

Якщо Xcode - улюблене середовище розробки Apple , то Visual Studio - це середовище розробки від Microsoft. Visual Studio в основному відома для створення програмного забезпечення, що включають в себе .NET. Це повний набір інструментів, що дозволяє зробити точну налагодження і налаштування програми. Є як Community-версія, так і PRO.




Eclipse

How to install

Eclipse - ще одна проста у використанні і при цьому потужна IDE для C і C ++.
Між Eclipse і NetBeans мало відмінностей, але все ж Eclipse є швидшою середовищем розробки.
Плюси Eclipse:

  • простота установки і використання;
  • безкоштовний і відкритий джерело;
  • підтримка декількох платформ;
  • потужний движок створення GUI-інтерфейсів;
  • віддалений системний провідник.
