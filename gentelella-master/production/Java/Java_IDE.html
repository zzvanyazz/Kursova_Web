Eclipse

How to install
У цій IDE найбільше співтовариство розробників, звідси ж і саме значна кількість плагінів. Зворотний бік - багато з них сумнівної якості та змісту. Корисна ця система ще й як інструмент мультимовної розробки - кількість підтримуваних мов завдяки ентузіастам перевищує 20. Крім того, це одна з кращих середовищ java розробки для android.



IntelliJ IDEA

How to install
Одна з найбільш функціональних середовищ для java розробки, оснащена системою інтелектуальної допомоги в написанні коду. Виходячи з контексту, IDEA налаштовує роботу автодоповнення і доступність інструментів. Велика кількість інструментів дозволяє прискорити розробку, наприклад за допомогою шаблонів і повторень, а також збільшити продуктивність кінцевої програми. Величезна кількість плагінів і надбудов під будь-яке завдання роблять середу java розробки IDEA майже ідеальним інструментом ..


Android Studio

How to install
Професійне середовище java розробки на Android. В основі лежить IntelliJ IDEA, оптимізована під рішення конкретних мобільних завдань: рефакторінга, збірки, шаблонізаціі, пошуку причин низької продуктивності і її збільшення.


BlueJ

How to install
Середовище розробки для тих, хто тільки робить свої перші кроки в програмуванні, тобто школярів і студентів. Тут простий інтерфейс, мінімум функцій і майже немає налаштувань, зате є версія середовища java розробки російською мовою.
